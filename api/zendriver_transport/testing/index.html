Test-only helpers — gated behind cfg(any(test, feature = "testing")).

Provides MockConnection, a paired pseudo-Chrome that lets downstream tests drive a real Connection without spawning a WebSocket. The mock and the connection share an in-memory duplex pipe built on the same DriverStream plumbing used by this crate’s internal actor tests.
