Window / viewport tools — browser_get_window, browser_set_window.

Wraps zendriver::Tab’s window controls (window_bounds, set_window_bounds, set_window_size, maximize, minimize, fullscreen). Geometry fields are device-independent pixels; Chrome omits geometry for a minimized window, so every field on WindowBoundsDto is optional.
