Source
Expand description

Navigation handlers — browser_goto, _back, _forward, _reload, _wait_for_idle.

All five tools share the crate::tools::common::current_tab helper and (for the URL-changing variants) a NavOutput return type that reports the post-navigation url + title plus an optional rendered HTML snapshot.
