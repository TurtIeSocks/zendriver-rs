Resolve, download (on cache miss), and return the path to a runnable Chrome-for-Testing binary. See module docs for version / channel precedence.

Does not need a browser open — the fetcher is independent of the SessionState’s zendriver::Browser slot. The state argument is kept for signature symmetry with every other tool handler.
