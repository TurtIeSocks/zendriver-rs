Module common

Expand description

Shared helpers used by multiple tool modules.

  • EmptyInput — JSON-schema-bearing placeholder for tools that take no arguments.
  • current_tab — resolves the current zendriver::Tab from a locked SessionState. Returns an owned (cheap-to-clone) handle so callers don’t fight the borrow checker against Browser::tabs(&self).await’s Vec<Tab>.
