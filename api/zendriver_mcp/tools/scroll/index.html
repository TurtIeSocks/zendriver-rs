Page-scroll tool — browser_scroll.

Wraps zendriver::Tab::scroll_with. The lib’s ScrollOptions follows the inverted CDP gesture convention (a negative yDistance scrolls the page down); this tool exposes intuitive screen axes (+dy = down, +dx = right) and negates both before dispatch, so the wire API reads naturally.
