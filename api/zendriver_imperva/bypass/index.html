Imperva bypass driver.

Public entry is ImpervaBypass — constructed via Tab::imperva() (zendriver crate, feature-gated). Single-struct dispatch: one wait_for_clearance runs the surface-aware poll loop, the optional ImpervaBypass::with_interception hook enables a Fetch-domain fast-path, and ImpervaBypass::on_captcha plugs a caller-supplied solver into the CAPTCHA escalation path. See module docs of crate::detection for surface inference rules.
