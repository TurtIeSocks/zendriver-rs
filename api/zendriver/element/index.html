Module element

Expand description

Handle to a DOM node via CDP RemoteObjectId / BackendNodeId.

Element is the result of a crate::Tab::find / crate::Tab::find_all / crate::Element::find query (or any of the traversal helpers). Actions sit on submodules:

  • actions — click / hover / focus / scroll / set_value / clear / upload_files.
  • input — type_text / type_text_fast / press / press_with.
  • reads — attribute access, innerText, outerHTML, bounding box, visibility / enabled state.
  • traversal — parent / nth_child.
  • isolated_eval — true isolated-world evaluate (with the element bound as el).
  • screenshot — element-scoped PNG capture.
  • refresh — auto-refresh-on-stale-handle support.
tab.goto("https://example.com").await?;
let h1 = tab.find().css("h1").one().await?;
assert_eq!(h1.inner_text().await?, "Example Domain");
