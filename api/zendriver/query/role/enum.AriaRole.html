Expand description

An ARIA role for use with crate::query::FindBuilder::role.

Covers the common WAI-ARIA roles. For roles not enumerated here, use AriaRole::Other with a &'static str of the role name.

§Examples

use zendriver::AriaRole;
assert_eq!(AriaRole::Button.to_css(), r#"[role="button"]"#);

pub fn to_css(self) -> String

Compile to a CSS attribute selector.

Tag-implicit roles (e.g. <button> implies role=button) are NOT auto-included; users querying by role get only explicit [role="..."] matches. This avoids surprising matches against tag-implicit elements that may have different accessibility behavior.

§Examples
use zendriver::AriaRole;
assert_eq!(AriaRole::Button.to_css(), r#"[role="button"]"#);
assert_eq!(AriaRole::Other("tooltip").to_css(), r#"[role="tooltip"]"#);
