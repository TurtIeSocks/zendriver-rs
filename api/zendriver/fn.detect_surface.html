Imperva WAF / Incapsula bypass surface re-exports.

Gated by the imperva cargo feature. The driver lives in the zendriver-imperva sub-crate; these aliases let downstream code reach it via the parent crate without an extra dependency. Entry point is Tab::imperva. Surface-only probe. Convenience for callers wanting a non-blocking “which surface is showing” check without the full snapshot.

use zendriver_imperva::{ImpervaSurface, detect_surface};

match detect_surface(tab).await? {
    ImpervaSurface::None => println!("clean page"),
    other => println!("imperva surface: {other:?}"),
}
