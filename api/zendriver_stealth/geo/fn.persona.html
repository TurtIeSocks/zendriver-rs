pub fn persona(country: Country) -> Persona
Expand description

Build a Persona overlay carrying a coherent locale + languages + representative timezone for country. Default locale policy: the country’s dominant language only, formed as lang-COUNTRY with its base subtag (e.g. CH -> de-CH + ["de-CH","de"]). The timezone is a single representative IANA zone per country (zone1970.tab first-occurrence, with curated overrides for large multi-zone countries, e.g. RU -> Europe/Moscow rather than Europe/Kaliningrad) — for genuinely multi-timezone countries this is an approximation, not the visitor’s actual zone; set Persona.timezone explicitly (.persona(..) / .persona_overlay(..)) when precision matters. An unknown country yields an empty overlay and a warning — never locks a value.
